Este repositorio es un laboratorio práctico (sandbox) diseñado para experimentar con herramientas de infraestructura como código (IaC), contenedores, automatización y CI/CD en un entorno **Serverless** sobre AWS utilizando **AWS Lambda** junto con imágenes de contenedor **Docker**.

El proyecto consta de una aplicación web construida con **FastAPI** que interactúa con una base de datos **PostgreSQL** para registrar el número de visitas, dockerizada y empaquetada como imagen de contenedor para ser desplegada en **AWS Lambda** mediante **Terraform** y configurada/automatizada con **Ansible** (en desarrollo).
